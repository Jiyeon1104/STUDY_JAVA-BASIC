# 쿠키
http 포로토콜의 비연결성과 비상태 유지 특성에 대한 문제점을 해결하는 방법 중 하나
쿠키를 사용하면 클라이언트의 상태 정보를 서버로 전송할 수 있어 서버가 클라이언트의 동일성을 확인할 수 있음

Cookie cookie = new Cookie("name", "value");

클라이언트가 서버에 서비스를 요청할 때 저장된 쿠키 정보를 요청 메세지의 헤더에 포함시켜 전송함
서버는 getCookies() 매서드로 쿠키 정보를 확인할 수 있음

# Session
세션과 쿠키의 가장 큰 차이는 정보를 서버사이드 (세션) 정보를 클라이언트 사이드 (쿠키) 즉 어디에 저장하느냐임
주로 세션은 로그인이나 장바구니 처리 등에 사용됨
